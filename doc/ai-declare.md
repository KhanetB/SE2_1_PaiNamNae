AI Declaration

ในการพัฒนาโปรเจกต์นี้ ได้มีการใช้เครื่องมือปัญญาประดิษฐ์ (AI) เพื่อสนับสนุนกระบวนการพัฒนาในขอบเขตดังต่อไปนี้:

การพัฒนาฟังก์ชันระดับ API (REVIEW Service)
ใช้ AI เพื่อช่วยในการ:

ออกแบบโครงสร้าง API

เขียนตัวอย่างโค้ดสำหรับ Review Service

จัดรูปแบบ Controller / Service / Route

แนะนำแนวทาง validation และ error handling

ทั้งนี้ ผู้พัฒนาได้ทำการตรวจสอบ แก้ไข และทดสอบโค้ดด้วยตนเองก่อนนำไปใช้งานจริง

การจัดทำเอกสาร (Documentation)
ใช้ AI เพื่อช่วย:

สร้างเอกสาร API จาก Source Code

สรุปคำอธิบายการทำงานของแต่ละ endpoint

จัดรูปแบบเอกสารให้อยู่ในรูปแบบที่เป็นทางการและอ่านเข้าใจง่าย

เนื้อหาทั้งหมดได้รับการตรวจสอบความถูกต้องและปรับแก้โดยผู้พัฒนา
